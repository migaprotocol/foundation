## Overview

MIGA Foundation is the community DAO site explaining the mission, governance, and transparency of the MIGA project - a Freedom of Information initiative funding anti-censorship technology for Iran.

## Mission

Fund and deploy technology that ensures free flow of information:
- Anti-censorship tools
- Secure communication infrastructure
- Open internet access
- Privacy-preserving technology

## Governance

MIGA operates as a fully transparent DAO:
- **1 MIGA = 1 Vote** in all treasury decisions
- On-chain treasury with public verification
- Community-driven proposal system
- No central authority

## Development

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev

# Build for production
pnpm build
```

## Tech Stack

- React 18 + Vite + TypeScript
- Tailwind CSS
- Deployed on GitHub Pages
